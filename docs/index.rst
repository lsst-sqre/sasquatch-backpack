##################
sasquatch-backpack
##################

Collects data and feeds sasquatch.

Install sasquatch-backpack from PyPI:

.. code-block:: bash

   pip install sasquatch-backpack

sasquatch-backpack is developed on GitHub at https://github.com/lsst-sqre/sasquatch-backpack.
